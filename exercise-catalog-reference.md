# IronLog Exercise Catalog Reference

Last reviewed: 2026-06-14

This file documents the exercise-to-muscle mapping used by IronLog's effective-set catalog. It is meant to be auditable, not perfect biomechanics.

## How The Numbers Work

- A primary target muscle receives `1.0` effective set per logged set.
- Secondary muscles receive partial credit from `0.1` to `0.75`.
- The source material below supports which muscles are primary or secondary movers. The exact decimal weights are IronLog heuristics, not direct EMG percentages.
- Heuristic scale:
  - `0.65-0.75`: very large secondary contribution.
  - `0.45-0.55`: strong secondary contribution.
  - `0.25-0.40`: moderate secondary contribution.
  - `0.10-0.20`: small support, grip, stability, or carryover.
- For squats, hamstring credit is intentionally low (`0.1`) because squat evidence suggests quads/glutes are much stronger hypertrophy targets than hamstrings.
